#The password generator.
A simple and modern Password Generator built using HTML, CSS, and JavaScript.
This app allows users to generate strong, secure passwords based on selected preferences such as length, numbers, symbols, uppercase letters, and more.

Tech Stack :-)

HTML5 – structure

CSS3 – styling & layout

JavaScript (Vanilla JS) – password generation logic.

#How It Works ?

-User selects password length

-User chooses which character types to include

-JavaScript creates a random password based on selected rules

-User can copy the generated password with one click.
